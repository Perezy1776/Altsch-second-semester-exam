BRIEF DESCRIPTION OF THE PROJECT

This project focuses on the development of a simple, static website using HTML syntax only. The website consists of a single page describing this work and a little biograpy about the work owner, all designed to provide essential information in a clean and structured format. Once the website is built, it is deployed on a Linux server using Apache2, a popular open-source web server software. The deployment process involves configuring the Apache2 server, transferring the website files to the appropriate directory, and ensuring the server is correctly set up to serve the website to users via a web browser. This project demonstrates the basics of website creation and hosting, highlighting skills in web design using HTML as well as server management and deployment using Apache2 on a Linux platform.
